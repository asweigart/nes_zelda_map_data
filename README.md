nes_zelda_map_data
==================

A project to derive the map data for the original NES Legend of Zelda game so that it can be used in hobbyist projects.

More info is here:
http://inventwithpython.com/blog/2012/12/10/8-bit-nes-legend-of-zelda-map-data/
